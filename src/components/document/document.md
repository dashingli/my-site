# Avater 组件(头像)

## 预览

![头像组件](/src/components/document/avaterPreview.png)

## 参数

| 参数  | 类型   | 是否必填 | 是否有默认值 | 参数意义               |
| ----- | ------ | -------- | ------------ | ---------------------- |
| url   | String | 必填     | 无默认值     | 头像地址               |
| width | Number | 非必填   | 150          | 设置头像宽度(宽高相等) |

## 启动测试

```bash
npm run test:Avater
```

# Icon 组件(图标)

## 参数

| 参数 | 类型   | 是否必填 | 是否有默认值 | 参数意义                   |
